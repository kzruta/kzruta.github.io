1 Вёрстку страницы блога начнём с подключения шрифтов. Необходимые шрифты возьмём в базе шрифтов Google Fonts. На странице используется шрифт Open Sans обычного, полужирного и жирного начертания (400, 600, 700). Помимо обычной стилизации шрифта, нам понадобится стилизация курсивом — italic. Для подключения шрифта добавим тег link в заголовок сайта.

2 Для определения параметров шрифта по умолчанию будем опираться обычный текст в статье. Размер шрифта для текста — 16px, высота строки — 24px. В качестве веб-безопасного шрифта используем Arial (шрифт без засечек). Полученные значения параметров шрифта зададим для тега body. Эти значения унаследуют большинство текстовых элементов, в том числе и обычный текст в статье.

3 Добавим разметку и сеточные стили для шапки страницы. Логотип — это изображение, обёрнутое в ссылку, главная навигация — список ссылок, телефон горячей линии — это ссылка вида a href="tel:...".

4 Добавим декоративное оформление для главной навигации — тень блока, используем CSS-свойство box-shadow. Параметры для тени берём в макете.

5 Разберём стилизацию ссылок главной навигации по сайту. Для начала определим параметры шрифта: размер, цвет, трансформации. Данные берём в макете. Размер шрифта — 13px, высота строки — 24px, цвет текста — #000000, к тексту применяется трансформация — все буквы в названии пунктов меню прописные — text-tranform: uppercase. В макете также задан межбуквенный интервал — 0.13em. Ссылки в обычном состоянии без подчёркивания — text-decoration: none.

6 Ссылка на текущую страницу блога имеет свои стили — перечёркнутый текст. Такая ссылка не содержит атрибут href и это позволит использовать селектор вида .site-navigation-link:not([href]). Для создания стиля применим CSS-свойство text-decoration: line-through.

7 Определим стили для состояний :hover, :active и :focus ссылки с логотипом сайта.

8 Определим стили для состояний ссылок главной навигации. При наведении и выборе с клавиатуры добавим для ссылки подчёркивание и немного уменьшим непрозрачность, при нажатии — ещё сильнее уменьшим непрозрачность. Для задания подчёркивания используем свойство border-bottom. Чтобы стили не применялись к ссылке для текущей страницы, в селекторе CSS-правила добавим атрибут href.

9 Изображение для ссылки будет фоновым, поэтому для добавления картинки используем CSS-свойство background. Эти стили определим для класса social-item-link-instagram, который указан вторым в атрибуте class у ссылки. Убираем заполнение фона цветом.

10 Декоративное оформление для телефона горячей линии добавим с помощью псевдоэлемента ::before. На макете изображения телефона немного повёрнуто, применим CSS-свойство transform: rotate.

11 Определим стили для состояний ссылки телефона горячей линии.

12 Переходим к основному содержимому страницы. Начнём с добавления хлебных крошек — это список ссылок. Добавим элементы в разметку. Зададим сеточные стили для списка.

13 Определим параметры шрифта для текста ссылок. Задаём размер и цвет текста, убираем подчёркивание.

14 Декоративный разделитель для адреса страницы в хлебных крошках добавим с помощью псевдоэлемента ::after. Псевдоэлемент будет только у ссылок с атрибутом href. Установим размеры, фоновое изображение и внешние отступы для декоративного элемента.

15 Определим стили для состояний наведения, нажатия и выбора с клавиатуры для ссылок с атрибутом href.

16 Приступим к разметке статьи «Отдых: как не превратиться в „загнанную лошадь“». Разметим метаданные. Для даты публикации используем тег time, автора статьи обернём в тег cite, количество просмотров и комментариев размечаем тегами span. Для заголовка статьи используем тег h1 — это главный заголовок страницы. Добавляем контентное изображение. В макете метаданные, заголовок статьи и изображение объединяются в один блок. В разметке для этого случая используем тег header, который далее стилизуем соответствующим образом.

17 Добавляем стили для шапки статьи. Задаём сеточные стили, заливку для фона.

18 Установим параметры шрифта для заголовка статьи. Указываем цвет, размер, жирность и межбуквенный интервал. С помощью трансформации текста преобразуем буквы в прописные. Для выравнивания текста по центру используем свойство text-align: center. С помощью CSS-свойства border-top добавим декоративную линию над заголовком.

19 Добавим сетки в метаданные. Декоративную линию, разделяющую метаданные в строке, сделаем с помощью CSS-свойства border-right.

20 Установим сеточные стили и параметры шрифта для метаданных. Укажем размеры, цвет и стиль шрифта.
