21 Добавим декоративные элементы для количества просмотров и сообщений. Для этого используем псевдоэлемент ::before.

22 Добавим сеточные стили для изображения в заголовке главной статьи сайта.

23 Под заголовком главной статьи размещена цитата с декоративным оформлением. Как обычно, начнём с разметки. Для этого используем тег blockquote. Текст цитаты поместим в тег p, источник обернём в cite.

24 Приступим к стилизации элемента blockquote class="main-article-quote". Установим сеточные стили, зададим цвет фона и фоновое изображение, укажем скругление границ блока.

25 Декоративные полоски на полях цитаты добавим с помощью псевдоэлементов ::before и ::after.

26 Добавим сеточные стили и выравнивание по центру для содержимого цитаты.

27 Определим параметры шрифта для текста цитаты. Зададим размер шрифта, курсивное начертание — font-style: italic.

28 Определим параметры шрифта для источника цитаты.

29 Если внимательно посмотреть на макет, то можно увидеть что всё текстовое содержимое главной статьи вписывается в область шириной 780px, которая выравнивается по центру макета. Для этого добавим ещё один контейнер div class="content-wrapper" и определим стили для него.

30 Следующий за цитатой абзац — это лид — вступительный текст к основной статье. Разметим лид тегом p. Добавим класс main-article-lead. Текст лида имеет полужирное начертание, реализовать это можно двумя способами: либо с помощью стилей, либо обернуть текст в тег b. Выберем второй вариант, он более универсальный.

31 Зададим параметры шрифта для лида и вложенного в него тега b.

32 Добавим стили для ссылок лида. Определим цвет и размер шрифта.

33 Определим стили для состояний ссылок.

34 Текст «Так каким же должен быть эффективный отдых?» разметим заголовком второго уровня.

35 Добавим стили для этого уровня заголовков. Определим размер шрифта, жирность, цвет.

36 Добавим изображение на страницу, это изображение контентное.

37 Добавим сеточные стили для изображения.

38 Следующий текст также можно отнести к лидам. На этот раз это текст с обычным начертанием, но нам пригодились стили предыдущего лида, так как мы предусмотрительно добавили полужирное начертание с помощью тега b.

39 Приступим к вёрстке списка советов. Добавим в разметку тег ul и li для первого элемента. Используем неупорядоченный список поскольку нам неважен порядок советов. Нумерация в этом случае служит лишь интересным декором. Несмотря на то что элементы списка советов различаются, в них есть общие составляющие — заголовок и текст, а значит отдельный совет обернём в тег section.

40 Стилизуем список. Избавимся от стилей по умолчанию list-style: none. Для создания нумерации используем CSS-счётчик. Инициализируем счётчик с помощью CSS-свойства counter-reset: advice-counter. За счёт counter-increment: advice-counter увеличиваем счётчик для каждого элемента списка. Номер будем формировать используя CSS-свойство content у псевдоэлемента ::before — content: "0" counter(advice-counter). Псевдоэлементы «вырваны» из потока. Также для них устанавливаем размеры, фоновый цвет и параметры шрифта.
