21 Следующие за шапкой два абзаца текста смело можно отнести к лидам. Это вступительный текст к основной статье. С помощью тега p добавим текст в разметку. Добавим класс main-article-lead.

22 Зададим параметры шрифта для лидов. С помощью text-align: justify выровняем текст в абзацах по ширине.

23 Возможно вы обратили внимание, что все блоки с текстом главной статьи имеют ширину 705px и расположены по центру страницы. Добиться такого поведения можно, если задать соответствующие правило для тегов p в элементе с классом main-article. Для следующего CSS-правила укажем селектор: .main-article p. В этом правиле зададим свойство width: 705px. Для выравнивания используем автоматический внешний отступ margin: 0 auto.

24 Далее в макете идёт цитата. При оформлении цитаты используются нестандартные кавычки. Кавычки используются как фоновое изображение. Начнём с разметки цитаты. Для этого воспользуемся тегом blockquote. Текст цитаты поместим в тег p, источник обернём в cite.

25 Переходим к стилизации элемента. Установим сеточные стили и фоновое изображение для элемента blockquote class="quote"

26 Следующим шагом установим стили для текста цитаты. Зададим параметры шрифта: название, размер, жирность и межбуквенное расстояние. С помощью трансформации текста преобразуем буквы к прописным.

27 Декоративные кавычки в тексте цитаты добавим с помощью псевдоэлементов ::before и ::after. В этом месте позволим себе вольность и в вёрстке «исправим» макет. В макете использована одинаковая кавычка и в качестве и открывающей, и закрывающей. Правильнее будет сделать открывающую кавычку «перевёрнутой» (девятки) и расположить внизу. Символ кавычки укажем в свойстве content. Укажем размеры шрифта для кавычек, значения возьмём в макете. Зададим отступ от текста.

28 Определим размер шрифта и жирность для автора цитаты.

29 Добавим в разметку ещё один лид с предложением идти дальше. Стили для элемента мы уже определили.

30 Приступим к вёрстке основных разделов статьи. Речь идёт о схожих по структуре элементах с информацией про лучшие блоги о моде. Подробно разберём разметку для одного такого элемента (остальные размечаются аналогично). Элемент содержит фотографию блогера, подпись к фотографии, заголовок и текстовое описание. Для разметки элемента используем тег section class="fashionblog". Изображение обернём в тег figure. Для подписи к изображению используем тег figcaption. Заголовок разметим тегом h2. Текст разметим тегом p. Для выделения фрагментов текста используем тег b. В некоторых элементах в тексте встречаются ссылки, их разметим обычным образом — тегами a.

31 Устанавливаем сеточные стили для section class="fashionblog". У section есть декоративный элемент — полоска, её добавим с помощью CSS-свойства border-bottom.

32 Определим стили для тега figure и figcaption внутри раздела. Для подписи к изображению зададим размеры шрифта. Полупрозрачным текст сделаем с помощью CSS-свойства opacity.

33 Определим стили для заголовка раздела. Укажем параметры шрифта. В нашем случае используется шрифт Cormorant. Установим размер и жирность.

34 Текстовое описание выравнивается по ширине контейнера. Для этого установим text-align: justify. Укажем межбуквенный интервал.

35 Благодаря тегу b акцентный текст уже выделен, но в макете установлено другое значение жирности, поправим это.

36 Добавим стили для ссылок раздела. Определим цвет и размер шрифта.

37 Определим стили для состояний ссылок.

38 Добавляем в разметку другие схожие разделы статьи.

39 Разберёмся как добавить на страницу врезку с информацией, относящейся к основному контенту, но не включённой в материал. Начнём с элемента «Редактируйте свои фото фильтрами от presets…». Элемент разметим с помощью тега aside. Содержимое элемента состоит из ссылки, в которой тегом span выделен хештег. Для размещения врезки на странице нам понадобится дополнительная обёртка div class="fashionblog-with-aside".
