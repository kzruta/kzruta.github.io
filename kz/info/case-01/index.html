1 Вёрстку страницы гида по стилю и моде «Fashionista» начнём с подключения шрифтов. Необходимые шрифты возьмём в базе шрифтов Google Fonts. На странице используются шрифт Cormorant обычного и жирного начертания (400,700) и шрифт Raleway обычного и полужирного начертания (400,600). Для подключения шрифтов добавим тег в заголовок сайта.

2 Определим параметры шрифта для обычного текста сайта. Опираться будем на текст основного содержимого статьи про лучшие блоги о моде. Этот текст набран шрифтом Raleway, размер шрифта 18px, высота строки 27px. В качестве веб-безопасного шрифта используем Arial (шрифт без засечек). Полученные значения параметров шрифта зададим для тега . Эти значения унаследуют большинство тегов с текстовым содержимым, в том числе и текст статьи.

3 Добавим разметку и сеточные стили для шапки страницы. Главная навигация — это список ссылок, логотип — изображение, обёрнутое в ссылку, для социальных сетей используем вырожденный список ссылок из одного элемента.

4 Более детально разберём стилизацию ссылок главной навигации по сайту. Для начала определим параметры шрифта: размер, цвет, трансформации. Данные берём в макете. Размер шрифта — 14px, цвет текста — #000000, к тексту применяется трансформация — все буквы в названии пунктов меню прописные — text-tranform: uppercase. В макете также задан межбуквенный интервал — 0.1em. Ссылки в обычном состоянии без подчёркивания — text-decoration: none.

5 Определим стили для состояний ссылок главной навигации :hover, :active и :focus. В макете эти стили явно не указаны, поэтому у нас полная свобода действий. При наведении и выборе с клавиатуры добавим для ссылки подчёркивание, при нажатии — уменьшим непрозрачность. Для задания подчёркивания используем свойство border-bottom.

6 Определим стили для состояний ссылки с логотипом сайта. При наведении будем менять непрозрачность на значение 0.6, при нажатии непрозрачность установим в значение 0.3. Для выбора с клавиатуры зададим стили идентичные стилям в состоянии наведения.

7 Для стилизации ссылки на страницу в Instagram нам понадобятся стили для доступного скрытия элементов страницы. Определим стили для класса visually-hidden.

8 Приступим к добавлению изображения для ссылки. Поскольку ссылка с классом social-item-link — это флекс-элемент, мы можем установить для неё ширину и высоту. Чтобы убедиться, что наши значения применились, временно зададим цвет фона для ссылки background-color: gray.

9 Изображение для ссылки будет фоновым, поэтому для добавления картинки используем CSS-свойство background. Эти стили определим для класса social-item-link-instagram, который указан вторым в атрибуте class у ссылки. Убираем заполнение фона цветом.

10 Определим стили для ссылки в состоянии наведения, нажатия и переключения с клавиатуры.

11 Переходим к основному содержимому страницы. Начнём с добавления хлебных крошек — это список ссылок. Добавляем элементы в разметку. Задаём сеточные стили для списка.

12 Определяем параметры шрифта для текста ссылок. Задаём размер, цвет текста и межбуквенное расстояние, убираем подчёркивание. С помощью трансформации текста преобразуем буквы в прописные.

13 Декоративный разделитель для адреса страницы в хлебных крошках добавим с помощью псевдоэлемента ::after. Псевдоэлемент будет только у ссылок с атрибутом href. Установим внешние отступы для декоративного элемента.

14 Определим стили для состояний наведения, нажатия и выбора с клавиатуры.

15 Приступаем к разметке статьи «На кого стоит равняться по стилю». Для заголовка статьи используем тег h1 — это главный заголовок страницы. При разметке метаданных используем тег dl — список определений, так как метаданные состоят из списка пар: термин — определение. В макете заголовок статьи и метаданные объединяются в один блок с фоновым изображением и декоративными элементами. В разметке для этого случая мы можем использовать тег header, который далее стилизуем соответствующим образом.

16 Добавляем стили для шапки статьи. Задаём сеточные стили. Для фонового изображения определим CSS-свойства: background-position: top center, background-repeat: no-repeat, background-size: cover. Изображение для фона добавим через встроенные стили тега header. Так мы упростим задачу для контент-менеджера, который в разметке сможет установить для шапки статьи любое фоновое изображение без правки стилевого файла.

17 Помимо изображения, в шапке статьи есть и другие декоративные элементы, они находятся в левом верхнем и правом нижнем углах области с полупрозрачным фоном. Для них в разметку добавлен тег div class="main-article-header-wrapper". Определим сеточные стили для элемента, добавим полупрозрачный фон. С помощью CSS-свойства position: relative мы будем управлять положением декоративных элементов в системе координат контейнера. Подробнее о позиционировании читайте в интерактивных курсах.

18 Добавим декоративные элементы. Для этого используем псевдоэлементы ::before и ::after. Рамки добавим с помощью CSS-свойства border, положение установим с помощью абсолютного позиционирования и указания соответствующих координат left, top для первого декоративного элемента и right, bottom — для второго.

19 Установим параметры шрифта для заголовка статьи. Текст в заголовке набран шрифтом Cormorant. Этот шрифт с засечками, поэтому в качестве веб-безопасного шрифта укажем шрифт Times New Roman. Указываем цвет, размер, жирность и межбуквенный интервал. С помощью трансформации текста преобразуем буквы в прописные. Для выравнивания текста по центру используем свойство text-align: center.

20 Зададим сеточные стили и параметры шрифта для метаинформации. Для терминов установим тип выравнивания текста text-align: right. С помощью font-weight: 600 выделим текст терминов.
